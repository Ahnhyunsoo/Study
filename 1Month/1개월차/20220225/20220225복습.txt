Heap = 런타임에 할당

Data = 프로그램이 종료될 때 반환
Stack = 함수가 종료될 때 반환

지역변수 = 중괄호 안에 있는 변수값
지역변수 = Stack메모리에 저장  (지역변수 함수의 매개변수)
함수가 종료되면 기존값이 지워진다.

Stack = 후입 선출 

주소값은 아래에부터 위로 쌓인다.

전역변수 = 데이터영역에 할당된다.

정적변수 = static 명령어를 사용하며 데이터 영역에 할당된다. 지역변수지만 데이터영역에 할당된다.

외부접근 내부접근에 따라 static을 표기하거나 안하거나 나뉜다. 외부에서 접근가능한건 static생략 내부에서 접근가능한건 static 표기
