함수 : 데이터를 가공하는 기능 상태를 말한다.
함수를 제작하는 이유는 코드의 재사용성 때문이다. (매번 필요할 때 마다 새로 만들수는 없기 때문에 만들어놓고 불러서 사용함)


반환타입 함수이름 매개변수(인자값, 파라미터)
void 	Func	(void) 함수선언부 
{ // 함수 정의부(몸체)
}


함수 사용시 주의점 : 
1. 매개변수의 타입과 갯수가 일치해야 한다.
2. 반환 타입이 void가 아닌경우 반드시 return함수를 사용해야한다.
3. return 할 때 반환 타입이 일치해야한다.

함수를 위쪽에 선언을 해두면 함수를 사용하는 부분보다 정의부분이 아래에 있어도 사용가능하다.

32비트 최대가용 용량 = 4G
메인메모리를 복사 -> 가상메모리
윈도우가 메모리를 충돌하지않게 관리해준다.
가상메모리 표현 = 0x = 16진수
메인메모리에는 불변하는 고정주소값이 있다.
사본의 주소 = 16진수이다. 메인메모리랑 곂치면 안되기 때문 16진수로 표현
메인 메모리 = 2진수로 값을 저장하고 저장된 값을 cpu가 연산후에 출력해준다.
