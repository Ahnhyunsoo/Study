const = 상수화 시키는 문법

포인터변수에서 const의 위치에 따라 결과가 바뀐다.
const int* p = 주소가 가르키는 값을 변경불가
int* const p = 주소를 변경 불가 (상수 포인터 오로지 하나의 메모리 공간만 참조하는 포인터)
=============================================================
배열 = 같은 자료형의 타입으로 여러 변수를 선언하고자 할 때 같은 이름을 묶어서 선언하기 위한 문법
배열은 블록 단위 데이터이다.
배열의 특징
1. 배열의 이름은 배열 전체 메모리의 첫 번째 주소 값이다.
2. 연속된 메모리 공간에 저장이 된다.
3. 배열을 선언할 때 []인덱스연산자의 값은 반드시 상수를 사용해야 한다.
배열 인덱스 순서대로 초기화 값이 결정
선언한 배열의 크기만큼 초기화를 할 값이 없는 경우 자동으로 0 초기화 일어난다.
4. 인덱스연산자 안에 배열의 개수를 알려주지 않더라도 초기화하는 값이 있으면 컴파일러가 초기화하는 숫자만큼 배열을 생성한다.
